\hypertarget{build_autotoc_md20}{}\doxysubsubsection{Cross compilation}\label{build_autotoc_md20}
Use arm-\/linux-\/gcc or arm-\/linux-\/g++ to compile programs. Use normal compiler flags (for example -\/g for a GDB debug build).\hypertarget{build_autotoc_md21}{}\doxysubsubsection{Project makefile}\label{build_autotoc_md21}
It is easier to use a makefile for compiling. The current makefile is located at the root of the project. It handles cross compilation.\hypertarget{build_autotoc_md22}{}\doxysubsubsection{Launching on a remote target}\label{build_autotoc_md22}
Once SSH key setup is done, it is easy and quick to compile and run the program using a small script.


\begin{DoxyCode}{0}

\DoxyCodeLine{HOSTNAME="{}cm4"{}}

\DoxyCodeLine{then}
